# MANIFESTO

I do not see programming as a profession alone.  
I see it as architecture for thought.

Software is language.  
Systems are philosophy.  
Every bug reveals something hidden beneath reality.

My work exists somewhere between precision and chaos —
between engineering and emotion.

Digital art became a way to translate:
- distortion into meaning
- silence into texture
- emotion into atmosphere

I am interested in systems —
both computational and human.

The way memory behaves.  
The way identity mutates online.  
The way beauty survives corruption.

---

# PHILOSOPHY

I believe:
- creativity should disturb comfort
- art should leave scars
- aesthetics matter
- code should carry intention
- identity is something built, not discovered

Fashion, literature, and music are not separate interests to me.  
They are extensions of the same search:
a search for atmosphere, meaning, and expression.
